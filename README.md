# Surpise-Project

🔫 CS2 Skin Price Tracker

Compare CS2 skin prices across multiple marketplaces — all in one place.


💡 The Idea
If you've ever tried to sell or buy a CS2 skin, you know the struggle — you have to manually open Steam Market, then TM, then Skinport, then Waxpeer... just to figure out where the best deal is. It's slow, repetitive, and annoying.
CS2 Skin Price Tracker solves this.
It automatically pulls prices from multiple marketplaces and displays them side by side, so you can instantly see where a skin is cheapest to buy or most profitable to sell — without opening 5 tabs.

✨ What It Does

📊 Aggregates prices from Steam Market, Skinport, Market.csgo, Waxpeer, and DMarket
🔍 Search any skin and see all prices compared in a single table
💰 Highlights the best price automatically across all sources
🔄 Caches prices locally so the site is fast and doesn't spam external APIs
⏱️ Shows when prices were last updated so you always know how fresh the data is
